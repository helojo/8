    有时我们需要看models操作时对应的SQL语句，

    可以用如下方法查看---

    在django project中的settings文件尾部添加如下代码

     

     1 LOGGING = {
 2     'version': 1,
 3     'disable_existing_loggers': False,
 4     'handlers': {
 5         'console': {
 6             'class': 'logging.StreamHandler',
 7         },
 8     },
 9     'loggers': {
10         'django.db.backends': {
11             'handlers': ['console'],
12             'level': 'DEBUG' if DEBUG else 'INFO',
13         },
