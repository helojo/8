    Selenium模块

    1.安装selenium

    • python2:pip install selenium
    • python3:pip install selenium

    2.设置浏览器驱动

    解压后必须与浏览器的exe放置于同一文件夹中

    • 驱动必须和浏览器版本匹配且位置放置于浏览器exe文件的同目录下
