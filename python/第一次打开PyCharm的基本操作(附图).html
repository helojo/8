    • 第一次打开PyCharm的基本操作(附图)

    第一次打开PyCharm可能需要修改一些个性化和了解一些基本操作,有助于接下来的学习过程.(后续可能会更新)

    我的版本是64位的1.3

    1.换界面皮肤

    默认黑色的,不喜欢黑色皮肤可以换成白色的

    • File---Settings --- Appearance&Behavior---Appearance --- Theme 

     

    2.更改字体大小

    默认字体很小,看着不舒服,需要自行修改

    • File -> Settings ->  Editor -> Font -> Size

    如果想换字体可以在Font里更改

     

    3.撤销和重做

    要撤销

    • 按Ctrl+Z。

    要重做

    • 按Ctrl+Shift+Z。

    4.批量注释和批量缩进

     批量注释:先选中要注释的代码块再进行操作

    • 按Ctrl+/
    • 取消批量注释再按一遍

    批量缩进

    • 选中块以后Tab

    反向缩进

    • 选中块后shift + tab

    5.关于tab和空格的处理

    由于pycharm对tab和空格识别很严格,此次有坑容易报错.所以可以自行把tab修改成4个空格.

    file-->setting-->Editor-->Code Style-->python

    6.自动排版

    编写代码后，可以使用Ctrl+Alt+L快捷键进行排版。

    具体更改可以看Settings->Editor->Code Style

    7.搜索

    普通搜索类Ctrl+N

    Run运行栏里的搜索Ctrl+F
