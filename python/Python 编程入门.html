    下文为曾经的学员感受~

    我是蟒营原型班的学员, 在这之前参加过另一个Python入门班的课程并认真完成全程的, 自认为也算Python入门了的, 但当大妈蟒营原型班通告的时候, 第一时间便报名参加了.

    原因很简单, 和很多其他原型班的学员一样, 跟大妈接触过,很认同大妈独特的理念, 相信大妈的课程不同于其他市面上的灌输式教学, 学到的将不仅仅是编程知识, 也不仅仅是 Python.

    事实上的确如此.

    视野的开阔

    以前无论自学\看书\其他培训课, 接触到的不是语法就是数据结构, 或者工具. 零散的知识点和套路, 千篇一律而无趣.

    在蟒营原型班, 是一套精心规划的Python生态, 你会在不到2个月的时间里, 很清楚我学了可以怎么用, 高手都在怎么用. 各种新知识, 既够你跟面试官大侃特侃, 也能在工作中脱颖而出, 还可以迁移到其他方面的学习.

    当然, 这些新知识, 于你我还只是入门, 但蟒营教的"入门能力"能帮你我继续"深入"任何一个知识.

    授之以渔

    入门的标准千人千样, 曾经的我认为的"入门"可能也和一些读者一样, 觉得掌握了语言的基本特性\语法\工具链, 就算入门了. 现在我想法不太一样了.

    您可以仔细看下蟒营的宣传海报, 和自己心中的"入门"比对一下.

    如果您参加了课程, 会听到(每周例会/单约)和看到(一堆ISSUE COMMENTS)大妈的

    各种叨叨, 若能认真领会, 便会明白, 熏陶的是"入门的能力", 而不仅仅是"入门的知识".

    只有知道了"怎么入门", 才算入门. 这是我现在的想法. 这也是蟒营的一个特色, 真正的授之以渔, 学到了便能以不变应万变.

    入门的技巧

    这个就太多了, 但核心能力就那么几个, 课程中大妈会反复强调和引导的, 我想提对我影响比较大的一个: Fake it until make it;

    这个和不懂装懂还是有区别的, 有兴趣的同学可以看看这个Fake it till you make it和不懂装懂有矛盾吗？, 里面有个叶同学的回复挺有意思, 给出了中文解释: "持续装逼，直到真逼" 或者"持续作秀，直到习惯成自然", 很形象啊, 就是大话先说在前面, 后面我为了圆我的大话认真做到了, 这其实也是(低调做人)高调做事.

    经常, 因为怕(被别人/自己)打脸, (比如我)多是:

    • 先做再说, 成功了开心的宣扬一番或暗喜, 失败了窃喜(没人知道)

    • 或是, 想一想觉得做成的可能性不大就放弃了.

    这样自然很多事都做不成了:

    • 要么很多事情都在门外面徘徊, 永远入不了门,

    • 要么浅尝辄止, 入门而不深, 梧鼠五技而穷.

    领会了 Fake it until you make it, 在课程中看到有些同学从真正的(完全的编程盲)小白, 在经历初期的各种懵逼\沮丧\不前\不挠, 到中期的质变, 最终脱颖而出成为优秀学员, 还是蛮感慨的.

    入门能力的迁移

    今天你报名学的是 Python 入门, 但你人生肯定不止接触这一门语言, 工作中还会有 java, c# 等等, 蟒营的课程能不能对你之后学习其他语言, 甚至其他非计算机类的学习有帮助? 我觉得是的.

    学完后你可以考虑将学到的Python"入门能力"迁移到其他语言/技术等的自学学习. 比如我现在就在自学另一门语言, 套用的是大妈的蟒营课程体系~. 说白了, 语言千万种, 特性就那么几类, 掌握了一种, 其他触类旁通. 关键在能力/思维的培养和迁移.

    最后

    入门的路很多, 选择权在你, 你可以走, 跑, 乘火车, 搭飞机, 无论哪种, 核心在你自己, 起决定作用的是你认真和坚持的心. 没有这份心, 即便蟒营也没有通天之力.

    不过, 有些事情不试过, 永远不知道会怎样. 如果你心中有那么一小撮学编程的小火苗, 就来蟒营燃烧小宇宙吧, 找回自己的认真和坚持.
