    项目环境搭建

    一、创建django项目

    1.创建python虚拟环境

    在虚拟机上创建python虚拟环境，因为实际项目部署，实在linux

    mkvirtualenv -p /usr/bin/python3.6 tzproject

    2.创建django项目

    1.安装django

    选择安装django2.1.x最新的一个版本2.1.10

    pip install django==2.1.10

    2.创建django项目

    cd ~/project/        # cd到你想放django项目路径的地方
django-admin startproject tzproject
