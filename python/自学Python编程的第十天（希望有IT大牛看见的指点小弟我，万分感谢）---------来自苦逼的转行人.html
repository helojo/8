    今天学的内容是有关函数

    一.重要知识点
            函数:对代码块和功能的封装和定义
            语法:
                def 函数名(形参:写在函数声明的位置的变量叫形参):
                    函数体
                函数名(实参:在函数调用的时候给函数传递的值叫实参)

                函数名:命名规则和变量一样

            函数的返回值:
                return, 函数执行完毕. 不会执行后面逻辑
                    1. 如果函数中不写return返回None
                    2. 只写return 返回None
                    3. return 返回值. 返回一个值
                    4. return 值1, 值2,... 返回多个值. 调用方接收到的是元组
            参数:
                在函数执行的时候给函数传递的信息
                1. 形参:
                    1. 位置参数.
                    2. 默认值参数. 当调用的时候不给传值使用默认值
                    3. 混合使用. 顺序:　1. 位置参数, 2, 默认值参数
                2. 实参:
                    １.位置参数. 按照位置给形参赋值
                    2. 关键字参数. 按照名称给形参赋值
                    3. 混合参数. 先用位置参数, 再用关键字参数

    二.错误总结
        1.参数忘记设置
            f=open('../day09-code/a6.txt',mode='r',encoding='utf-8')
            s=f.readline()
            lst=s.split()
            # print(lst)
            result=[]                       -->>一开始这里忘记给一个参数来接收最后的结果
            for el in f:
                # print(el)
                ss=el.split()
                print(ss)
                dict={}
                for i in range(len(ss)):
                    dict[lst[i]]=ss[i]
                # print(dict)
                result.append(dict)         -->>这里忘记把迭代出的结果放到一起
            print(result)
        #出现参数忘记设置的原因是因为没仔细去考虑题目要有哪些变量
        解决方法:写程序时要先琢磨透要设置哪些变量,哪些不用设置.先把要求弄懂才开始写程序.
