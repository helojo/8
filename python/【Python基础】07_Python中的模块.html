    1.模块的概念

    • 模块 就好比 工具包，要想使用这个工具包中的工具，就需要 导入import 这个模块
    • 每一个以扩展名 .py 结尾的 Python源代码文件 都是一个 模块
    • 在模块中定义的 全局变量、函数 都是模块能够提供给外界直接使用的工具

    注：

    可以在模块中定义 变量或者函数

    模块名也是一个标识符

    2.模块的使用

    import 导入模块，模块名.变量/模块名.函数
