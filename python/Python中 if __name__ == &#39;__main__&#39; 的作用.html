    Python文件可以直接运行，也可以 import 到其它文件中使用

    if __name__ == '__main__' 就是控制代码在这两种情况下的执行过程

     

     

    每个Python模块都包含内置变量，直接运行的时候，name 等于 main ，代表自身

     

     

     

     

    __name__，当模块被调用执行的时候，name等于文件名
