    tensorflow-build

    tensorflow 源码编译，提升硬件加速，支持cpu加速指令,suport SSE4.1、SSE4.2、AVX、AVX2、FMA,ver:1.1.0rc1、1.4.0rc1、1.14.0-rc1、2.0.0b1

    最近使用过程中发现老是报avx指令问题，于是编译了几个重要的版本，欢迎测试，平台是mbp 2017
    另外2.0应该是全网比较少了，欢迎下载测试，编译时间太长了，差不多12h，风扇呼呼的大转，不建议再做第二次。
