    • vscod如何自定义 python虚拟环境
    原文出处:本文由博客园博主feiquan提供。
    原文连接:https://www.cnblogs.com/feiquan/p/11442873.html

    参考文档：https://code.visualstudio.com/docs/python/environments

    1.创建虚拟环境，cd到当前目录

    py -3 -m venv env

    2.Ctrl+Shift+P，选择Python：Select Interpreter，选中刚才新建的env环境，即可自动定位。然后Ctrl+`来创建一个新的Terminal窗口，发现自动进入了虚拟环境，不用运行activated脚本来运行虚拟环境了。

     3.如果有错误的话，可以手动指定，在.vscode文件夹中，修改settings.json文件，添加一下设置即可，其中env为.vscode同级目录下的虚拟环境所在的文件夹。

    {
    "python.pythonPath": "env\\Scripts\\python.exe"
}
