    写完后就可以开始打包成软件了

    这里我就介绍下如何用 pyInstaller模块 打包成软件

    首先

    在命令管理器即cmd中pip install pyInstaller安装pyInstaller模块

    安装完成之后在命令行中输入pyInstaller -F 文件路径安装

    -F表示打包成单一个文件没有-F则会打包成一个文件夹和软件

    pyInstaller -F  D:\python\1.py

    打包完成后软件的目录在这里

    把这个路径复制到文件夹目录既可以找到软件
