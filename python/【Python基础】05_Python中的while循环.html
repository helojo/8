    　　条件满足时，做的事情2

    　　……

    注：while语句以及缩进部分是一个 完整的代码块

    4.break 和 continue

    • break 某一条件满足时，退出循环，不再执行后续重复的代码
    • continue 某一条件满足时，不退出循环，不执行后续重复的代码

    注：break 和 continue 只针对当前所在循环有效

    5.循环嵌套
